# Telegram Mini App — Нумерологический калькулятор

Четыре расчёта: 
1) Число дня рождения — `numerology.json`
2) Число жизненного пути — `life_path_numbers.json`
3) Квадрат Пифагора — `pifagor.json`
4) Ритм на год — `rhythm_year.json` (год берётся текущий: 2025)

## Запуск
```bash
npm i
npm run dev
# или pnpm i && pnpm dev
```

Откройте `http://localhost:5173`

## Подключение к Telegram Mini App
- В боте укажите URL сборки (`npm run build`: папка `dist/` на любой хостинг).
- Используется автотема Telegram через CSS-переменные.

## Логика расчётов
- **День рождения:** ищем, в каком разделе `numerology.json` указан введённый день (в массиве `dates`). Если не нашли — приводим день к одной цифре.
- **Жизненный путь:** сумма цифр ДД + ММ + ГГГГ, редуцируем до 1..9.
- **Пифагор:** считаем количество каждой цифры 1..9 в строке `DDMMYYYY`, строим матрицу 3×3 в порядке [1,4,7; 2,5,8; 3,6,9]. Выводим базовый текст и уровень из `pifagor.json` (ключ уровня: `X`, `1`, `11`, `111` и т.п.).
- **Ритм на год:** личный год = сумма цифр дня + месяца + целевого года, редуцируем до 1..9. Если составное число одно из `13/14/15/21/22` — используем его как ключ для расширенной расшифровки.

## Где лежат расшифровки
`public/data/*.json` — это ваши исходные файлы (подгружаются по запросу при раскрытии карточки).

## Стек
Vite + React + TypeScript, Zustand. Без серверной части.
